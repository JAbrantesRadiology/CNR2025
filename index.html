Sobre o Workshop

Bem-vindo ao workshop "Deep Learning aplicado à Radiologia: O meu primeiro modelo de IA". Este workshop foi projetado para residentes de radiologia com pouca ou nenhuma experiência em programação, bem como profissionais interessados na aplicação de IA em imagens médicas.

Durante este workshop prático de 2 horas, você aprenderá os fundamentos do deep learning e desenvolverá seu primeiro modelo de IA para classificação de pneumonia usando radiografias de tórax.

Utilizaremos o Google Colab, uma plataforma gratuita baseada em nuvem, que permite escrever e executar código Python sem a necessidade de configurar um ambiente de desenvolvimento local.

Agenda do Workshop

Introdução (15 minutos)

Breve visão geral da IA em radiologia

Importância da detecção de pneumonia usando radiografias de tórax

Objetivos do workshop

Exploração de Dados (15 minutos)

Introdução ao conjunto de dados (RSNA Pneumonia Detection Challenge)

Compreensão dos formatos de dados e anotações

Técnicas básicas de visualização de dados

Desenvolvimento do Modelo (45 minutos)

Construção de uma rede neural convolucional (CNN) usando TensorFlow/Keras

Treinamento do modelo no conjunto de dados

Avaliação do desempenho do modelo usando métricas apropriadas

Interpretação e Discussão (30 minutos)

Análise dos resultados do modelo

Compreensão das limitações e possíveis vieses

Discussão sobre aplicações no mundo real e próximos passos

Perguntas e Encerramento (15 minutos)

Responder às perguntas dos participantes

Fornecer recursos para aprendizado adicional

Coletar feedback

Notebooks do Google Colab

Notebook 1

Carregamento e Pré-processamento de Dados

Aprenda a carregar o conjunto de dados RSNA Pneumonia Detection Challenge, explorar suas características e preparar os dados para treinamento.

Abrir no Colab

Notebook 2

Construção e Treinamento do Modelo

Desenvolva uma rede neural convolucional (CNN) usando TensorFlow/Keras e treine-a para classificar radiografias de tórax com e sem pneumonia.

Abrir no Colab

Notebook 3

Avaliação e Interpretação dos Resultados

Avalie o desempenho do seu modelo, visualize os resultados e compreenda como interpretar as previsões em um contexto clínico.

Abrir no Colab
