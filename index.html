Sobre o Workshop

Bem-vindo ao workshop "Deep Learning aplicado à Radiologia: O meu primeiro modelo de IA". Este workshop foi projetado para internos e especialistas de radiologia com pouca ou nenhuma experiência em programação, bem como profissionais interessados na aplicação de IA em imagens médicas.

Durante este workshop prático de 2 horas, iremos aprender os fundamentos do deep learning e desenvolver o primeiro modelo de IA para classificação de pneumonia usando radiografias de tórax.

Utilizaremos o Google Colab, uma plataforma gratuita baseada em nuvem, que permite escrever e executar código Python sem a necessidade de configurar um ambiente de desenvolvimento local.

Notebooks do Google Colab

Notebook 1

Carregamento e Pré-processamento de Dados

Aprenda a carregar o conjunto de dados RSNA Pneumonia Detection Challenge, explorar as suas características e preparar os dados para o treino

Abrir no Colab

Notebook 2

Construção e Treino do Modelo

Desenvolva uma rede neural convolucional (CNN) e treine-a para classificar radiografias de tórax com e sem pneumonia.

Abrir no Colab

Notebook 3

Avaliação e Interpretação dos Resultados

Avalie o desempenho do seu modelo, visualize os resultados e compreenda como interpretar as previsões no contexto clínico.
